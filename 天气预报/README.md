# 🌈 天气预报 (Weather Forecast)

一款基于和风天气 API 开发的、具有“可爱清新”设计风格的现代化天气预报 Web 应用。该项目旨在通过直观、美观且动态的界面，为用户提供实时的天气信息、未来 7 天预报及贴心的生活建议。

## ✨ 核心特性

- 📍 **精准定位与搜索**：支持“城市+区名”的精细化搜索（如：深圳宝安），精准获取街道级别的天气数据。
- 🌡️ **实时状态监控**：直观展示当前气温、体感温度、湿度、风力风向及小时级降雨概率。
- 📅 **7 天天气预报**：清晰呈现未来一周的天气趋势。
- 💡 **智能生活助手**：根据实时气象数据提供个性化的穿衣建议、紫外线防范提醒及天气预警。
- 🎨 **治愈系视觉设计**：采用玻璃拟态（Glassmorphism）布局，配合可爱的动态天气图标，提升交互愉悦感。

## 🛠️ 技术栈介绍

本项目采用了现代化的前端开发方案，强调**无框架依赖（Vanilla JS）**的高性能与灵活性。

### 1. 核心层 (Core)
- **HTML5**: 使用语义化标签构建结构，确保页面逻辑清晰且利于 SEO。
- **JavaScript (ES6+)**: 原生驱动，负责所有 API 交互（Fetch API）、DOM 动态渲染、搜索逻辑及核心天气建议算法。

### 2. 界面与样式 (UI & Styling)
- **CSS3 (Vanilla)**:
    - **玻璃拟态 (Glassmorphism)**: 运用 `backdrop-filter: blur()` 与半透明背景，打造极具通透感的“玻璃感”容器。
    - **弹性布局 (Flexbox/Grid)**: 确保多端自适应展示，在移动端与桌面端均有出色表现。
    - **动态微交互**: 使用 `CSS Keyframes` 实现图标浮动效果及加载动画。
- **Google Fonts**: 自定义字体方案，使用了 `ZCOOL KuaiLe` (站酷快乐体)、`Fredoka` 等，营造亲和力强的视觉氛围。
- **Lucide Icons**: 引入矢量化图标库，通过 `lucide.createIcons()` 动态生成精致的交互图标。

### 3. 数据服务 (Data Services)
- **和风天气 (QWeather) API**: 集成了实时天气、7 天预报、生活指数（逐天/逐小时）及地理位置搜索等全方位数据接口。
- **Fetch API**: 采用异步请求机制，确保数据获取过程不阻塞界面渲染。

## 🚀 快速开始

1. **克隆项目**:
    ```bash
    git clone [repository-url]
    ```
2. **配置 API Key**:
    在 `app.js` 文件顶部，将 `API_KEY` 替换为您在 [和风天气开发者控制台](https://dev.qweather.com/) 申请的密钥：
    ```javascript
    const API_KEY = '您的_API_KEY';
    ```
3. **本地运行**:
    直接通过浏览器打开 `index.html` 即可开始使用。
